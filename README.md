# 🌸 Personal Portfolio Website

A modern, responsive personal portfolio built using **HTML, CSS, and Vanilla JavaScript**. This website showcases my skills, projects, and journey as a Frontend Developer and Computer Science student.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Light and dark mode with theme preference saved in local storage
* Animated typing effect in the hero section
* Scroll reveal animations using Intersection Observer
* Sticky navigation bar with active section highlighting
* About, Skills, Projects, and Contact sections
* Interactive cards, buttons, and hover effects
* Accessible and semantic HTML structure
* Contact form with client-side validation
* Smooth scrolling and modern UI design

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts

> **Note:** No frameworks or libraries have been used in this project. It was built entirely with HTML, CSS, and Vanilla JavaScript as part of my effort to strengthen my frontend fundamentals and gain a deeper understanding of core web technologies.
